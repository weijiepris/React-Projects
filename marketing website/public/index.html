Get Started

Ease of Access application

BigDocker is a lightweight docker image capable of running Big Data Architecture

Made with students in mind

We want to create an application that is easy to use for students who are new to big data and ease them into the process of learning big data

Easily Available

Our Project is easily accessible with a click of a button

Made with React and Sage Cells

The application will be dockerised and the UI will be intuitive using React as the main page

Instructions

1) Install Docker, we recommend Docker Desktop

2) Get our Docker-Compose YAML file

3) Move this YAML file to a folder of your choice.

4) Launch a new terminal and navigate to that folder.

5) Run docker compose up in the terminal.

6) When the terminal instance no-longer updates with new information, the application is ready to be used. Do not kill the terminal instance.

7) Launch the application by going to http://localhost:3000/

8) When done, issue the CTRL-C command in the same terminal you opened in step 4

9) Repeat steps 4-8 for future use.

About BigDocker

Bigdocker's objective is to create a docker image that has the ability to allow students and those who are trying to learn how to use all the basic big data architecture functions, with ease and without any hassle in installing separate programs.

Features

Below are some of the features we have implemented in BigDocker.

Downloadable with One Click

Our Application is lightweight as it is a all in one solution.

Step by step guide for students

A step by step guide will be at the sidebar to guide students through each step to ensure that they fully grasp big data concepts

Fast and easy to install

The students only need to download our application to use the full features on big data

Settings customised to the User's preference

The user would be able to change settings within the applications for a better viewing experience

Full statistical graphs available

The user would have full data visualization charts which can be all viewed in a single page

All user hosted

The user will host the docker server itself within their computer and it will utilise the computer's processing power to run it.
